# View Files

<Row>
  <Col>

  This endpoint is used to view all your files uploaded to Supavec.


  ## Request Body {{ tag: 'POST', label: '/user_files' }}

    <Properties>
      <Property name="pagination" type="object">
        Pagination parameters for the query.
        - `limit`: `integer (default: 10)` - The number of files to fetch.
        - `offset`: `integer (default: 0)` - The offset of the files to fetch.
      </Property>
      <Property name="order_dir" type="string">
        The direction to order the results by.
        Available options: `desc`, `asc`.
      </Property>
    </Properties>

  </Col>
  <Col>
    <Note>
      Before you can make requests to the Supavec API, you will need to grab **your**
      API key from [your dashboard](https://www.supavec.com/dashboard).
    </Note>

    ```ts {{ language: 'js', title: 'JavaScript' }}
    const res = await fetch("https://api.supavec.com/user_files", {
      method: "POST",
      headers: {
        "Content-Type": "application/json",
        authorization: apiKey,
      },
      body: JSON.stringify({ pagination, order_dir }),
    });
    ```
